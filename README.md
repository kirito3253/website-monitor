# 豊田高専 奨学金情報更新通知ボット (Toyota CT Scholarship Monitor)

豊田工業高等専門学校の「授業料・奨学金」ページを定期的に監視し、更新があった際に LINE で即座に通知する自動化ツールです。

## 🌟 開発背景
高専生活において非常に重要な奨学金情報は、更新のタイミングを逃すと申請期限に間に合わないリスクがあります。毎日手動でサイトを確認する手間を省き、情報の更新をいち早くキャッチするために開発しました。

## 🛠 使用技術
- **Language:** Python 3.10
- **Libraries:** - `Requests` (Web サイトのデータ取得・LINE API 通信)
  - `BeautifulSoup4` (HTML 解析・特定要素の抽出)
- **Automation:** GitHub Actions (24時間定期実行スケジュール)
- **API:** LINE Messaging API (プッシュ通知)
- **Security:** GitHub Secrets (API キー・ユーザー ID の隠匿)

## 🚀 主な機能
1. **特定エリアの監視:** サイト全体の更新ではなく、`<main>` タグ内の主要コンテンツに絞って監視することで、サイドバーの更新などによる誤通知（ノイズ）を防いでいます。
2. **ハッシュ値による比較:** 取得したコンテンツを MD5 ハッシュ値に変換して比較することで、効率的かつ確実に変更を検知します。
3. **完全自動運用:** GitHub Actions を利用し、1日3回（日本時間 9:00, 15:00, 21:00）自動でチェックを行います。
4. **セキュアな設計:** API トークンなどの機密情報は環境変数として管理し、コード内には直接記述していません。
